# Partiful first-run setup

Run this once with the operator before the first extraction. Afterwards the
signed-in browser session is all the skill needs.

## 1. Check for an existing session

Open `https://partiful.com` in the in-app browser. If the operator's profile
avatar shows in the navigation, they are signed in — skip to step 3.

## 2. Sign in or sign up

Send the operator to `https://partiful.com/login`. Partiful signs in with a
phone number and SMS code. The operator completes this themselves in the in-app
browser — never ask for, read, or enter their phone number or verification
code.

## 3. Get on the guest list

Guest-list visibility is controlled per event by the host, and usually requires
an RSVP. If the operator wants the roster for an event they have not responded
to, have them open the invite link and RSVP first — their choice which
response, but it must be theirs, not the skill's.

## 4. Verify access

Open the target event page. The roster is reachable when a `Guest List` section
with a `View all` control is visible. If it is not, the host hides the guest
list for this operator's account — report that plainly and do not work around
it.
