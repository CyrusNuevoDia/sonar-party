# Luma first-run setup

Run this once with the operator before the first extraction. Afterwards the
signed-in browser session is all the skill needs.

## 1. Check for an existing session

Open `https://luma.com` in the in-app browser. If the operator's avatar shows in
the top-right navigation, they are signed in — skip to step 3.

## 2. Sign in or sign up

Send the operator to `https://luma.com/signin`. Luma signs in with an email code
or Google. The operator completes this themselves in the in-app browser — never
ask for, read, or enter their credentials or verification codes.

## 3. Get on the guest list

Luma shows guests only to people the host allows — usually registered
attendees. If the operator wants the roster for an event they have not
registered for, have them open the event page and register first. If
registration needs approval or payment, stop and let the operator decide.

## 4. Verify access

Open the target event page. The roster is reachable when a visible guest button
such as `Aaron Francis, Demir Eren and 64 others` appears. If it does not, the
host hides the guest list for this operator's account — report that plainly and
do not work around it.
